Шаблон классa has_Begin_End.

Позволяет определить есть ли у типа переданного к качестве шаблонного параметра методы begin() и end()

Данные класса

template<typename T >
constexpr bool has_Begin_End< T >::value = std::is_same<void, decltype(detect(std::declval<T>()))>::value
static

Если у проверяемого типа есть методы begin() и end() то value - true.
